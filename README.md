Backend:
  - SQLite
  - .NET 6
  - Entity Framework 6
  - CQRS/Mediatr
  - xUnit
  
 Frontend:
  - Angular 13
  - Font Awesome
  - ESLint
  - Bootstrap 4
  - Jasmine
  
To run the app:

1) Pull code
2) Go to src/backend
  - dotnet restore
  - dotnet build
  - dotnet run
3) Go to src/frontend
  - yarn install
  - ng serve
4) Open your browser and go to localhost:4200
